# Manual Funcional v2

Este manual describe las nuevas funcionalidades desde la perspectiva del usuario final.

## Exportación del Ambiente de Trabajo
Para descargar todos los datos, logs y modelos asociados a un análisis, utilice el botón "Exportar Ambiente" en la interfaz. Se descargará un archivo ZIP con todos los artefactos.

## Validación de Datos
La nueva capa de validación se ejecuta automáticamente durante ciertos flujos de trabajo. Si sus datos no cumplen con las reglas de calidad predefinidas (ej. columnas faltantes, formato incorrecto), el sistema le notificará los errores encontrados.

## Mejoras de Seguridad
El sistema ahora es más robusto contra la carga de archivos maliciosos o demasiado grandes, mejorando la estabilidad y seguridad general de la plataforma.
